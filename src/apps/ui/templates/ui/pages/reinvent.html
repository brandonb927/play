Battlesnake is a fun way for developers to practice new technologies.

Write Code

Battlesnake developers build and deploy a web server that implements the Official Battlesnake API. You can use any language and any technology you want - all you need to play is a web server that can receive and respond to HTTP requests.

Compete Online

Create games against your friends and colleagues, track wins and losses, and climb the ranks of the Battlesnake Arena. We also regularly host online and live events with hundreds of participants.
