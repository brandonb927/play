Developers build and deploy a web server that responds to the Official Battlesnake API. You can use any language and technologies you want - you just need a web server that can receive and respond to HTTP requests. Once your web server is running create your snake and start playing games.

Compete in Online Events

You can create Battlesnake games against other developers, track wins and losses, and climb the ranks of the Battlesnake Arena. We also regularly host online and live events with hundreds of participants.
